# ☕ Ngopikuy

Aplikasi CRUD Menu Cafe berbasis Laravel.

## 📋 Fitur:
- Login dan Register untuk Admin
- Tambah Menu, Edit, Hapus (CRUD)
- Validasi Nama & Harga
- Bootstrap Styling bawaan Laravel UI

## 🛠️ Teknologi:
- Laravel 12
- Laravel UI (auth + bootstrap)
- PHP 8.2
- MySQL (via `.env`)
